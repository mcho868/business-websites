# business-websites

뉴질랜드 한인 소상공인을 위한 비즈니스 웹사이트 제작 서비스 랜딩 페이지.

Deploys to `business-websites.manseungchoi.com`.

## Stack

- [@arrow-js/core](https://www.arrow-js.com/) — 리액티브 템플릿 (No React)
- [Vite](https://vitejs.dev/) — 개발 서버 · 번들러
- Vanilla CSS

## 개발

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
npm run preview
```
