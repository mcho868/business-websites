Build a lightweight, production-quality landing page for my business website service using ArrowJS.

Tech stack requirements
- Use ArrowJS as the main UI framework
- Keep the implementation as lightweight as possible
- Prefer `@arrow-js/core` for the page UI and reactivity
- Do not build this as a heavy React SPA
- If any React package is used, keep it minimal and only for a very specific reason
- Avoid unnecessary dependencies, large UI libraries, animation libraries, and bloated packages
- Prioritize fast load time, small bundle size, and clean maintainable code

Project goal
- Create a conversion-focused service landing page for small business website clients
- This page is specifically for people who want a business website built
- The audience is mainly Korean small business owners in New Zealand
- The page should feel trustworthy, simple, modern, and easy to understand
- This is not a resume page and not a general developer portfolio page
